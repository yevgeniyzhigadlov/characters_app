# Rick & Morty Characters (Flutter, BLoC)

Flutter-приложение, реализованное в рамках технического задания.  
Загружает список персонажей из публичного API, поддерживает офлайн-доступ и управление состоянием через BLoC.

---

## Функциональность

- Список персонажей из Rick & Morty API
- Пагинация с подгрузкой при прокрутке
- Обновление списка через pull-to-refresh
- Кеширование загруженных данных для офлайн-доступа
- Добавление и удаление персонажей из избранного
- Экран избранных с сортировкой:
    - по имени
    - по статусу
    - по виду
    - по количеству эпизодов
- Светлая и тёмная тема с сохранением выбора
- Локализация интерфейса (RU / EN)
- Обработка ошибок и пустых состояний
